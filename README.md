#Ringo's Bounty Hunter Portfolio

Welcome to Ringo's Portfolio, a creative web project that blends storytelling, interactive design, and immersive features to showcase the life and work of Ringo, a seasoned bounty hunter.

🚀 Features

Gunshot Sound Effects: Every click triggers realistic gunshot audio, adding an immersive Wild West atmosphere.

Animated Transitions: Smooth animations when navigating between pages, enhancing the storytelling experience.

Profile Drawers: Interactive sections that reveal detailed profiles of individuals Ringo has hunted or is currently hunting.

Responsive Design: Fully optimized for small phones, tablets, laptops, and desktops with tailored layouts for each screen size.

Dynamic Navigation: Active page highlighting in the menu for clear orientation.

Slideshow Galleries: Visual showcases of Ringo’s missions and hunts with interactive controls.

Custom Cursor: Crosshair-style cursor for thematic immersion.

📂 Project Structure

Index.html / Index.css → Homepage introducing Ringo and his services.

about-me.html / about-me.css → Detailed biography, vision, and rivalries.

portfolio.html / portfolio.css → Showcase of hunts, missions, and interactive slideshows.

contact.html / contact.css → Contact details, callback form, and terms.

input-form.html / input-form.css → User form for hiring Ringo or leaving reviews.

terms-conditions.html / terms-conditions.css → Legal terms for engaging Ringo’s services.

ActivePage.js → Highlights the current page in the navigation.

MobileMenu.js → Handles mobile menu toggling.

slideshow.js → Controls slideshow functionality.

Sounds.js → Manages gunshot sound effects.

styles.css → Global styles, fonts, and cursor customization.

🎨 Design Highlights

Radial and Linear Gradients: Used extensively for backgrounds to create dramatic, thematic visuals.

Animations: Keyframes for expanding sections, fading text, and background transitions.

Grid Layouts: Modular and responsive grids for structured content presentation.
